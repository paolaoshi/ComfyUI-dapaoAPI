# 🎬 Sora2 视频生成节点使用说明

## 📖 节点简介

**Sora2 视频生成节点** 是一个强大的视频生成工具，支持**文生视频**和**图生视频**两种模式，可以根据文字描述或输入图像生成高质量视频内容。

### ✨ 核心特性

- 🎬 **文生视频**：根据文字描述直接生成视频
- 🖼️ **图生视频**：基于输入图像生成动态视频
- 📐 **多种分辨率**：支持 16:9、9:16、1:1、21:9、4:3 等宽高比
- ⏱️ **时长可调**：支持 5秒、10秒、15秒、20秒
- 🎨 **高清模式**：可选择开启高清画质
- 💧 **水印控制**：可选择显示或隐藏水印
- 🔒 **私密模式**：保护您的创作隐私
- 🔔 **回调通知**：支持任务完成后的回调通知

---

## 🚀 快速开始

### 1️⃣ 配置 API 密钥

#### 方法一：在配置文件中设置（推荐）

编辑 `sora2_config.json` 文件：

```json
{
    "api_key": "YOUR_API_KEY_HERE",
    "base_url": "https://api.example.com",
    "timeout": 300,
    "max_retries": 3
}
```

#### 方法二：在节点参数中设置

在节点的 `🔑 API密钥` 参数中直接输入您的 API Key。

> **注意**：请将 `base_url` 替换为实际的 Sora2 API 地址。

---

## 📝 参数说明

### 必填参数

| 参数 | 类型 | 说明 | 默认值 |
|------|------|------|--------|
| **📝 提示词** | 文本 | 视频内容描述 | "一只可爱的小猫在草地上奔跑，阳光明媚，高清画质" |
| **🔑 API密钥** | 文本 | Sora2 API 密钥 | 留空则使用配置文件 |
| **📐 宽高比** | 选择 | 视频宽高比 | 16:9 |
| **⏱️ 时长(秒)** | 选择 | 视频时长 | 10 |
| **🎨 高清模式** | 布尔 | 是否开启高清 | 开启 |
| **💧 水印** | 布尔 | 是否显示水印 | 显示 |
| **🔒 私密模式** | 布尔 | 是否开启私密 | 开启 |

### 可选参数

| 参数 | 类型 | 说明 |
|------|------|------|
| **🖼️ 输入图像** | IMAGE | 图生视频的输入图像（留空则为文生视频） |
| **🔔 回调地址** | 文本 | 任务完成后的回调 URL |

## 🎯 使用场景

### 场景一：文生视频

**适用于**：根据创意文字描述生成视频

**操作步骤**：
1. 在 `📝 提示词` 中输入详细的视频描述
2. 选择合适的 `📐 宽高比` 和 `⏱️ 时长`
3. 不连接 `🖼️ 输入图像` 参数
4. 点击执行

**示例提示词**：
```
一只可爱的橘猫在阳光明媚的草地上奔跑，背景是蓝天白云，
镜头跟随猫咪移动，画面温馨治愈，电影级画质
```

### 场景二：图生视频

**适用于**：让静态图像动起来

**操作步骤**：
1. 连接一个图像节点到 `🖼️ 输入图像`
2. 在 `📝 提示词` 中描述期望的动态效果
3. 选择合适的参数
4. 点击执行

**示例提示词**：
```
让图片中的人物微笑并挥手，背景保持不变，
动作自然流畅，画面稳定
```

## 📐 宽高比选择指南

| 宽高比 | 适用场景 | 说明 |
|--------|----------|------|
| **16:9** | 横屏视频 | 适合 YouTube、B站等平台 |
| **9:16** | 竖屏视频 | 适合抖音、快手等短视频平台 |
| **1:1** | 方形视频 | 适合 Instagram 等社交媒体 |
| **21:9** | 超宽屏 | 适合电影级视觉效果 |
| **4:3** | 传统比例 | 适合经典风格内容 |

---

## ⏱️ 时长建议

| 时长 | 适用场景 | 说明 |
|------|----------|------|
| **5秒** | 快速预览 | 测试效果，节省时间 |
| **10秒** | 短视频 | 适合大多数场景 |
| **15秒** | 中等内容 | 适合完整表达一个动作 |
| **20秒** | 长内容 | 适合复杂场景或故事 |

---

## 🎨 高清模式说明

- **开启高清**：生成更高分辨率的视频，画质更好，但生成时间更长
- **关闭高清**：生成速度更快，适合快速预览

## 💡 提示词编写技巧

### 文生视频提示词结构

```
[主体] + [动作] + [环境] + [镜头] + [风格]
```

**示例**：
```
一位年轻女孩（主体）在海边跳舞（动作），
夕阳西下，海浪拍打沙滩（环境），
镜头缓慢推进（镜头），
电影级画质，温暖色调（风格）
```

### 图生视频提示词结构

```
[动作描述] + [保持元素] + [动态效果]
```

**示例**：
```
人物微笑并向镜头挥手（动作描述），
背景和服装保持不变（保持元素），
动作自然流畅，画面稳定（动态效果）
```

## 🔄 工作流程

1. **提交任务** → 节点返回 `任务ID`
2. **异步生成** → Sora2 API 在后台生成视频
3. **查询结果** → 使用任务ID查询生成状态
4. **下载视频** → 获取视频下载链接

> **注意**：本节点只负责提交任务，不会等待视频生成完成。您需要使用任务ID通过其他方式查询生成结果。

## 📊 输出说明

### 输出 1：任务ID

- **类型**：STRING
- **说明**：用于查询视频生成状态的唯一标识符
- **用途**：保存此ID，稍后用于查询视频下载链接

### 输出 2：详细信息

- **类型**：STRING
- **说明**：包含任务提交的详细信息
- **内容**：
  - 提示词
  - 参数设置
  - 任务ID
  - 提交耗时
  - 使用提示

## ⚠️ 注意事项

1. **API 地址**：请确保 `sora2_config.json` 中的 `base_url` 配置正确
2. **API 密钥**：请妥善保管您的 API Key，不要泄露
3. **生成时间**：视频生成通常需要几分钟，请耐心等待
4. **任务ID**：请保存好任务ID，用于后续查询
5. **网络连接**：确保网络连接稳定，避免请求失败
6. **图像格式**：输入图像会自动转换为 PNG 格式并进行 base64 编码

## 🐛 常见问题

### Q1: 提示 "API密钥未配置"

**解决方法**：
- 检查 `sora2_config.json` 文件是否存在
- 确认 `api_key` 字段是否填写
- 或在节点参数中直接输入 API Key

### Q2: 提示 "API 请求失败"

**解决方法**：
- 检查网络连接是否正常
- 确认 API 地址是否正确
- 检查 API Key 是否有效
- 查看控制台日志获取详细错误信息

### Q3: 提示 "请求超时"

**解决方法**：
- 检查网络连接速度
- 增加 `sora2_config.json` 中的 `timeout` 值
- 稍后重试

### Q4: 如何查询视频生成结果？

**解决方法**：
- 保存节点返回的任务ID
- 使用 Sora2 API 的查询接口查询任务状态
- 或设置回调地址，等待任务完成通知
