# Seedream 4.0 文生图节点使用说明

## 📖 节点简介

Seedream 4.0 文生图节点是基于火山引擎 Seedream 4.0 API 的文本生成图像节点，支持丰富的参数配置和风格预设。

## 🎯 主要功能

- ✅ 文本描述生成高质量图像
- ✅ 支持多种风格预设（写实、动漫、2.5D等）
- ✅ 灵活的分辨率和宽高比控制
- ✅ 种子控制（固定/随机/递增）
- ✅ 批量生成多张图片
- ✅ 负向提示词支持

## 📝 参数说明

### 必填参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| **提示词 (prompt)** | STRING | 描述您想要生成的图像内容 |
| **API Key** | STRING | Seedream API 密钥 |
| **端点 ID (endpoint_id)** | STRING | Seedream 模型端点ID |

### 可选参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| **负向提示词 (negative_prompt)** | STRING | 空 | 描述不想出现的元素 |
| **宽度 (width)** | INT | 1024 | 图像宽度（512-2048） |
| **高度 (height)** | INT | 1024 | 图像高度（512-2048） |
| **推理步数 (steps)** | INT | 20 | 生成步数（1-50），越高质量越好但速度越慢 |
| **引导系数 (cfg_scale)** | FLOAT | 7.0 | 提示词引导强度（1.0-20.0） |
| **出图数量 (image_count)** | INT | 1 | 单次生成图片数量（1-4） |
| **风格预设 (style_preset)** | DROPDOWN | 无 | 预设风格选项 |
| **种子控制 (seed_control)** | DROPDOWN | 随机 | 种子生成方式 |
| **种子值 (seed)** | INT | 0 | 固定种子值（种子控制为"固定"时使用） |

## 🎨 风格预设说明

| 风格名称 | 适用场景 |
|----------|----------|
| **写实照片** | 真实摄影风格 |
| **数字艺术** | 现代数字绘画 |
| **动漫风格** | 日系动漫画风 |
| **2.5D风格** | 介于2D和3D之间 |
| **电影感** | 电影级渲染效果 |
| **梦幻风格** | 奇幻梦境效果 |

## 🎲 种子控制说明

| 控制方式 | 说明 | 适用场景 |
|----------|------|----------|
| **随机** | 每次生成使用随机种子 | 探索多样化结果 |
| **固定** | 使用指定的固定种子值 | 复现特定结果 |
| **递增** | 种子值自动+1 | 批量生成连续变化 |

## 💡 使用技巧

### 1. 提示词编写建议
```
好的提示词示例：
"一只可爱的橘猫，坐在阳光明媚的窗台上，毛发柔软蓬松，眼神温柔，高清摄影，浅景深"

避免的写法：
"猫" （过于简单）
```

### 2. 分辨率选择
- **正方形 (1024×1024)**：通用场景
- **横版 (1280×720)**：风景、横幅
- **竖版 (720×1280)**：人像、海报

### 3. 参数组合建议
- **高质量输出**：steps=30, cfg_scale=8.0
- **快速预览**：steps=15, cfg_scale=6.0
- **精确控制**：固定种子 + 高引导系数

### 4. 批量生成
- 设置 `image_count` 为 2-4
- 配合 `递增` 种子控制
- 可快速生成系列图片

## ⚠️ 注意事项

1. **API 配额**：每次生成会消耗 API 额度，批量生成时注意控制数量
2. **生成时间**：高分辨率 + 高步数会增加生成时间（约30-60秒）
3. **提示词语言**：建议使用中文或英文，其他语言可能效果不佳
4. **负向提示词**：API 支持情况以官方文档为准

## 🔧 配置文件

如果不想每次都输入 API Key 和 Endpoint ID，可以在 `config.json` 中配置：

```json
{
    "api_key": "你的API密钥",
    "endpoint_id": "你的端点ID",
    "base_url": "https://ark.cn-beijing.volces.com/api/v3",
    "timeout": 120
}
```
